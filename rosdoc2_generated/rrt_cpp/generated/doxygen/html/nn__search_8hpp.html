Checks if a point is already inserted in the samples vector with a certain tolerance eps_L2_squared. Fills both the X_near vector with the points that within a ball of radius sqrt(delta_R_squared) centered at coord and the closest_point in samples wrt to coord if fill_fn_params is set to true.

Parameters
samples
coordThe point to check if it is already inserted in samples.
X_nearWhen the function returns false, it contains the points in samples within delta_R centered at coord if fill_fn_params is set to true; otherwise, it is undefined.
closest_pointWhen the function returns false, it contains the closest point in samples to coord if fill_fn_params is set to true; otherwise, it is undefined.
fill_fn_params
eps_L2_squaredSquared Euclidean distance tolerance.
delta_R_squaredSquared radius of the ball centered at coord.
Returns
true if coord is too close to any point in samples with a tolerance of eps_L2_squared; otherwise, false.
