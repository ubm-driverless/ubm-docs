Detailed Description

ROS2 node that implements the Pure Pursuit algorithm for controlling a vehicle along a raceline.

The PathFollowerNode class is a ROS2 node that subscribes to odometry and pose messages, and publishes Ackermann drive commands to control a vehicle along a raceline. The node implements a PD cpntroller for steering or the stanley controller, and uses feedforward action for steering control. The node also supports dynamic parameter updates for tuning the controller gains and other parameters.
