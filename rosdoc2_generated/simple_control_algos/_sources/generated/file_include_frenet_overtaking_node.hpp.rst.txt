Detailed Description
--------------------

The :ref:`exhale_class_classFrenetOvertakingNode` class is a ROS2 node that subscribes to odometry, pose, and laser scan messages, and publishes Ackermann drive commands to control a vehicle along a raceline. The node implements the Frenet Optimal Trajectory algorithm for steering control, and uses a PD controller for speed control. The node also supports dynamic parameter updates for tuning the controller gains and other parameters.
