FrenetOptimalTrajectory(float max_road_width = 1.0, float d_road_w = 0.5, float dt = 0.2, float max_t = 2.0, float min_t = 1.0, float target_speed = 1.0, float d_t_s = 0.1, float robot_radius = 0.4, float k_j = 0.1, float k_t = 0.1, float k_d = 1.0, float min_look_ahead = 0.0, float look_ahead_gain = 0.0, float speed_decay_factor = 0.95, float min_speed_reduction_gain = 0.1)

Constructor for the FrenetOptimalTrajectory class. Initializes parameters for calculating the optimal trajectory in Frenet coordinates.

Parameters:

  • max_speed – Maximum allowable speed.

  • max_accel – Maximum allowable acceleration.

  • max_curvature – Maximum allowable curvature.

  • max_road_width – Maximum width of the road for lateral movement.

  • d_road_w – Lateral step size for road width sampling.

  • dt – Time step for trajectory generation.

  • max_t – Maximum prediction time.

  • min_t – Minimum prediction time.

  • target_speed – Desired speed of the vehicle.

  • d_t_s – Longitudinal speed step size for sampling.

  • n_s_sample – Number of speed samples.

  • robot_radius – Radius of the robot for collision checking.

  • k_j – Weight factor for jerk cost.

  • k_t – Weight factor for time cost.

  • k_d – Weight factor for distance cost.

  • k_lat – Weight factor for lateral cost.

  • k_lon – Weight factor for longitudinal cost.

  • min_look_ahead – Minimum look-ahead distance

  • look_ahead_gain – Proportional gain to speed for look-ahead

  • speed_decay_factor – Speed decay factor for path not found

  • min_speed_reduction_gain – Minimum speed reduction gain for path not found

Plans the optimal Frenet path based on current vehicle state and obstacles.

Parameters:

  • csp – CubicSpline instance representing the reference path.

  • s0 – Initial longitudinal position on the path.

  • c_speed – Current speed of the vehicle.

  • c_d – Initial lateral displacement from the reference path.

  • ob – List of obstacles as tuples (x, y) in global coordinates.

Returns:

Tuple containing the optimal Frenet path and x and y position of target point.
