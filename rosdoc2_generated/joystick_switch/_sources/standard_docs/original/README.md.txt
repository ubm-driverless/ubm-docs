## Node Purpose

This node is a switch that sends joystick commands to either /ego_joy or /opp_joy.
To switch between these two topics you need to press the button R2.
This is usefull if you want to control 2 cars with only 1 joystick.
For example you can start an algorithm on one car and then manually control the other.

## Topics

Subscribes to /joy and publish to /ego_joy and /opp_joy
