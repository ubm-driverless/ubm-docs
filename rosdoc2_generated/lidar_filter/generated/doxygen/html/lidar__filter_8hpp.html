Detailed Description

Header file for lidar signal filtering utilities.

This header file defines several utilities for processing and filtering lidar data. It includes methods to apply basic signal processing operations such as linear spacing, mean and median convolutions, and outlier removal. These functions are designed to work with 1D arrays of floats representing lidar signal data. The filters aim to smooth the data, remove noise, and handle outliers, making the lidar data more reliable for further processing.

The filtering operations include:

  • Linearly spaced values generation with central_linspace
  • Mean convolution for smoothing with mean_convolution and mean_convolution_box
  • Median filtering for noise reduction with median_filter
  • Outlier removal with outliers_filtering
